## HolocronRouter

The HolocronRouter included in this repository is **permanent, immutable, and canonical**.  
It serves as the **universal routing registry** for the Unified GS1 Resolver Mesh.

- Fully open-source and permissionless
- Deployed at the same deterministic address on all supported chains via CREATE2
- Provides integrity, availability, and auditability guarantees
- All resolution operations in the resolver mesh rely on this contract
- **No further updates will be made to this contract**
